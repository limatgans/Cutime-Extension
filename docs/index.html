Why Common Units?

The International System of Units is great, but broken. When it was written, the authors let some inconsistency exist as part of the specification for convenience. The SI system was an epic feat when it was published in 1960. However, today, as some of us look back and say,

"Why is time 60 seconds in a minute instead of 100?",
"Why is 11:50 not midway from 11 to lunchtime?",

it might be worth thinking of an alternative system - one set entirely in base 10.
